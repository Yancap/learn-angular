Property Binding

Property Binding são usados para definir propriedades e atributos nas tags, com duas formas 

  • [propriedade]=”valor”
  • propriedade=”{{valor}/}”

Event Binding

Event Binding são feitos utilizando o nome entre parênteses (evento), sendo possível passar o parâmetro ”event” para função 

(event)=”funcao($event)”
